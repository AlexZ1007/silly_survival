# <p align="center">SillySurvive</p>

##  Tema

**SillySurvive** este un joc 3D de tip **survival**, în care jucătorul trebuie să colecteze resurse, să crafteze obiecte și să depășească diverse provocări pentru a supraviețui și a progresa în joc.


##  Rulare Proiect

> ℹ️ Este necesară versiunea **Unity 6000.0.59f2 (LTS)**.

1. Clonează repository-ul (cu suport pentru Git LFS):
   ```bash
   git lfs install
   git clone https://github.com/AlexZ1007/silly_survival.git
2. Deschide Unity Hub și adaugă proiectul folosind **Add project from disk**.

3. Deschide scena principală din: **Assets/Scenes**
